下一步可能的功能方向

基于"复习 + 整理 + 输出"的工作流，选出你最想先做的 1~2 个

A

闪卡复习模式

把笔记自动转成问答题卡片，复习时先显示问题，再回放对应视频片段揭晓答案。支持 Anki 导出。

B

标签 + 知识脑图

给笔记打 #标签，自动生成按标签聚合的时间线或简易脑图，发现知识点之间的关联。

C

精彩片段导出

选中一条或多条笔记，一键导出对应的视频片段（GIF / MP4），附带笔记文字，方便分享。

D

学习报告生成

按视频生成结构化学习摘要：章节大纲 + 关键截图 + 笔记整理，导出为 Markdown / PDF。
