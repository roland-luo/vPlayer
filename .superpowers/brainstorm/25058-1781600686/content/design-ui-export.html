## 01. Tokio Runtime

### [00:03:42] 创建 tokio runtime 的两种写法

一种是使用 `#[tokio::main]` 宏：

```rust
#[tokio::main]
async fn main() {
    // ...
}
```

另一种是手动创建 Runtime：

```rust
let rt = tokio::runtime::Runtime::new().unwrap();
rt.block_on(async {
    // ...
});
```

![screenshot](assets/00_03_42.png)

### [00:08:15] await 和 block_on 的区别

- `await` 必须在 async 函数内使用
- `block_on` 会阻塞当前线程直到 future 完成
